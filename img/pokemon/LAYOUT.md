# Directory Layout

```
pokemon
|-- avatar          -- bigger images
\-- sprite          -- sprites
    |-- ani         --   animated
    |   |-- reg     --     regular coloring
    |   \-- shiny   --     shiny coloring
    \-- static      --   static
        |-- reg     --     regular coloring
        \-- shiny   --     shiny coloring
```

Filename: `pokemoName[-f].{png|gif|jpg}`
 * `-f` -- female pokemons
 * `gif` -- animated
 * `png|jpg` -- static
